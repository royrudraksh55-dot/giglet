# Giglet

Giglet is a web platform that connects clients who need small tasks completed with workers willing to perform them.

## Features
- User authentication
- Task posting
- Task browsing
- Task application system
- Task completion tracking

## Tech Stack
- React
- Vite
- Firebase
- Tailwind CSS

## Installation

Clone the repository

git clone https://github.com/royrudraksh55-dot/giglet.git

Install dependencies

npm install

Run development server

npm run dev
